JOIN : 테이블 2개 이상을 연결하여 검색
-CROSS JOIN(FROM절에 테이블 2개이상)
SELECT * FROM EMP WHERE ENAME='SCOTT';-1
SELECT * FROM DEPT;-2
1,2 결합해서
SELECT * FROM EMP, DEPT WHERE ENAME='SCOTT';
이렇게 출력하면 EMP의 1행 * DEPT의 4행,총4행이 출력된다

-EQUI JOIN
EMP E,DEPT D의 DEPTNO값이 일치되는 조건을
이용하는 JOIN
형식)FROEM EMP,DEPT WHERE E.DEPTNO=D.DEPTNO;

EX1)
급여가 2000이상인 직원만 이름, 업무(JOB), 급여, 부서명, 근무지 출력
SELECT ENAME,JOB,SAL,DNAME,LOC FROM EMP E,DEPT D
WHERE E.DMPTNO=D.DMPTNO AND SAL>2000;
여기서 DNAME,LOC은 DMPT 테이블에 있어서 FROM절에 추가해 줘야 된다

EX2)
LOC이 CHICAGO인 사람의 이름, 업무, 급여를 출력
SELECT ENAME,JOB,SAL FROM EMP E,DEPT D WHERE E.DEPTNO=D.DEPTNO
AND LOC='CHICAGO';(LOC DEPT 테이블이라 FROM절에 추가해 줘야 된다

EX3)
부서번호가 10 이거나 20번인 사원의 이름(ENAME), 업무(JOB), 근무지(LOC). 급여순
SELECT ENAME,JOB,LOC,SAL FROM EMP E,DEPT D
WHERE EMP E,DEPT D AND AND DEPTNO IN (10,20);
>>>>>WHERE 절에서 이렇게 코딩하면 오류가 난다.
DEPTNO가 EMP 소속인지 DEPT 소속인지 붙여줘야 한다.
DEPTNO>>>D,DEPTNO OR E.DEPTNO

EX4)
 COMM이 NULL이고 급여가 1200이상인 사원의 이름, 급여, 입사일, 부서번호, 부서명
부서명순, 급여큰순 정렬
SELECT ENAME,SAL,HIREDATE,DEPTNO,DNAME FROM EMP E,DEPT D WHERE
E.DEPTNO=D.DEPTNO AND COMM IS NULL ORER BY DNAME,SAL DESC;

-NON-EQUI JOIN
EMP와 SALGRADE가 JOIN한 경우
FROM EMP,SLAGRADE(등급정보가 들어있는 테이블)
SELECT * FROM EMP WHERE ENAME='SCOTT'; EMP의 1행
SELECT * FROM SALGRADE;>>급여 등급 정보를 알 수 있다. SALGRADE 5행
SELECT * FROM EMP, SALGRADE WHERE ENAME='SCOTT' AND SAL>=LOSAL AND SAL<=HISAL; 

EX1)
모든 사원의 사번, 이름, JOB, 상사사번, 급여, 급여등급(1등급, 2등급)
SELECT EMPNO, ENAME, JOB, MGR, SAL, GRADE||'등급' FROM EMP, SALGRADE
    WHERE SAL BETWEEN LOSAL AND HISAL;

EX2)
COMM이 NULL아 아닌 사원의 이름, 급여, 급여등급, 부서번호, 부서이름, 근무지
SELECT ENAME,SAL,GRADE,E.DEPTNO,DNAME,LOC 
FROM EMP E,DEPT D,SALGRADE 
WHERE  E.DEPTNO=E.DEPTNO AND NOT COMM IS NULL AND SAL BETWEEN LOSAL
AND HISAL;

EX3)
이름, 급여, 입사일, 급여등급, 부서명, 근무지
SELECT ENAME,SAL,HIREDATE,GRADE,DNAME,LOC
FROM EMP E,DEPT D,SALGRADE WHERE BETWEEN LOSAL AND HISAL;
