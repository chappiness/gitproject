3.선택자란?HTML 문서 중 특정 부분을 선택하여 해당 부분의 속성을
변경하는 목적으로 사용

4.*{ padding:0; margin:0;} 문서 전체에 밖여백 안여백 0으로 한다.

5.id(css를 이용해 속성변경할때 #을 붙여줘야 한다.)
div(css를 이용해 속성변경할때 .을 붙여줘야 한다.)

6.속성 선택자:tag의 속성을 선택하여 css의 속성을 설정
선택자(속성):지정한 속성을 가지고 있는 요소 찾아 스타이을 적용
선택자(속성=값):속성 값이 특정값과 같은 경우를 찾아 스타일 적용
선택자(속성^=값):속성 값이 지정한 문자로 시작하는 속성값에 스타일 적용
선택자(속성$=값):속성 값이 지정한 문자로 끝나는 속성에 대해서만 스타일 적용

7.후손(아들) 및 자손 선택자:tag의 속성을 선택하여 css의 속성을 설정

8.동위(형제) 선택자(+:하나만,-:다선택):태그의 동등한 위치를 판단하여 css 속성 설정

9.반응 선택자(hover):마우스를 올려 놓았을 때의 반응에 따른 속성을 설정

10.상태 선택자:상태에 따라 css 속성이 변화하는 설정

11.구조 선택자:구조에 따라 css 속성이 변화하는 설정

12.링크 선택자:문서에 링크되어 있는 문자를 선택하여 css 속성 설정

13.부정 선택자:나를 제외한 모든 태그에 css 속성 적용
