new:클래스로부터 객체를 생성시키는 연산자
클래스의 인스턴스화:클래스로부터 객체를 만드는 과정
객체:모든 인스턴스를 대표하는 포괄적인 의미 
"객체를 생성시켜주고 초기화 시켜놓는다."

메인에서 클래스에 있는 생성자함수를 불러낼때 쓰임
new 연산자로 생성된 객체는 메모리 영역에 생성된다.
객체를 생성시킨 후,객체의 주소를 리턴하도록 되어있다.
참조변수의 타입은 인스턴스의 타입과 일치해야 한다

클래스의 구성멤버
클래스,데이터,생성자(리턴이 없음).메소드

ex)super 클래스 생성자함수S 인스턴스 s
아들 클래스 생성자함수A 인스턴스 a
손자 클래스 생성자함수B 인스턴스 b
S s=new S S에서 인스턴스화 하면 그 주소가 S에 저장
B b=new B
C c=new C>>>>B c=new C 가능(사람은 포유류다)
C b=new B(포유류는 사람이다) 그래서 될수 없다.
따라서 손자클래스 new를 이용하여 배열로 표현이 가능하다.

일종의 작업지시서(기본 설계도)
-상수와 추상 메소드만 들어갈 수 있음 다시 말하면 객체 생성(인스턴스화 x)
cf)추상 클래스:한개 이상의 추상 메소드를 지닌 클래스

상속은 다중상속 x //A extends B (B<<<A)
인터페이스는 다중 구현이 가능 하다 A implements B,C
